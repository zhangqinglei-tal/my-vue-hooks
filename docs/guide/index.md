# 介绍

My Hooks 是一个 Vue 3 Composition API 工具库集合，提供了大量实用的组合式函数（hooks），帮助你更高效地开发 Vue 应用。

## 为什么选择 My Hooks？

- **Tree Shakeable**: 支持按需导入，只打包你使用的代码
- **TypeScript**: 完整的类型支持，提供更好的开发体验
- **轻量级**: 零依赖（除了 Vue），不会增加项目体积
- **易用**: 简单直观的 API 设计
- **文档完善**: 每个 hook 都有详细的文档和示例

## 设计理念

My Hooks 遵循以下设计原则：

1. **可组合性**: 每个 hook 都是独立的，可以轻松组合使用
2. **响应式**: 充分利用 Vue 3 的响应式系统
3. **类型安全**: 完整的 TypeScript 类型定义
4. **性能优化**: 支持 tree shaking，减少打包体积
